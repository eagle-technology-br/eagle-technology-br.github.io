Como funciona

Tecnologia

Arquitetura de ponta combinando visão computacional e aprendizado semi-supervisionado para máxima eficiência com dados clínicos reais.

01

YOLOv8

Arquitetura de detecção de objetos estado da arte, otimizada para imagens radiográficas de alta variabilidade.

02

Semi-supervised

Pipeline de aprendizado semi-supervisionado com pseudo-labels e threshold de confiança ≥ 0,85 para escalar o treinamento.

03

Active learning

Seleção inteligente das amostras mais incertas para anotação humana, reduzindo custo sem sacrificar qualidade.

04

MLflow + DVC

Rastreabilidade completa de experimentos e versões de dataset — atendendo aos requisitos da ANVISA RDC 657/2022.

05

LGPD compliant

Pipeline de anonimização DICOM auditável com log imutável, em conformidade com a Lei Geral de Proteção de Dados.

06

Caminho ANVISA

Desenvolvido desde o primeiro commit com os requisitos da RDC 657/2022 — ISO 13485, ISO 14971 e IEC 62304.

Onde estamos

Roadmap

Desenvolvimento estruturado em fases, com financiamento FAPESP PIPE e foco em registro regulatório.

Concluído

MVP funcional

Modelo YOLO treinado com 13.000 imagens públicas, capaz de identificar dentes, ausências e patologias básicas.

Em andamento

PIPE Fase 1 — FAPESP

Construção de dataset proprietário brasileiro (10.000 imagens), expansão para 32 classes e validação clínica com especialistas.

Próxima fase

Produto comercial

API de análise integrada a sistemas de gestão odontológica, modelo SaaS por exame analisado.

Longo prazo

Registro ANVISA

Submissão do dossier técnico regulatório e obtenção do registro como Software como Dispositivo Médico (SaMD) — RDC 657/2022.
